Básico de HTML5

Essa linguagem é voltada para o desenvolvimento web, de modo a estruturar todo seu conteúdo. Seu funcionamento básico é organizado em elementos, os quais podem ser formados por: tag de abertura e seus atributos, seu conteúdo e uma tag de fechamento. A partir disso, vamos aprender algumas das tags mais utilizadas:

  • tag head

    • Essa tag é utilizada para apresentar informações ao navegador, tais como metadados, isto é, dados sobre como os dados devem ser apresentados ao usuário. É aqui que o arquivo em CSS é referenciado, mas veremos isso mais pra frente...

  • tag body

    • A tag body é responsável por armazenar todo conteúdo visível ao usuário

  • tag header

    • Essa tag indica um cabeçalho de uma seção, usualmente ela contém títulos e/ou dados iniciais de um conteúdo

  • tag h1-h6

    • Responsável por determinar os níveis de título, sendo h1 o mais importante e, portanto, o maior e h6 o menos importante, logo, o menor

Essas são apenas algumas tags utilizadas em html, mais informações podem ser encontradas em cursos online disponíveis em plataformas como digitalinnovation.one ou até mesmo em sites como https://developer.mozilla.org/pt-BR/docs/Web/HTML/HTML5. Ademais, como esta página foi desenvolvida em HTML e CSS, basta apertar F12 ou clicar com o botão direito do mouse e selecionar a opção "inspecionar" para ter acesso ao código.

Básico de CSS

Seu nome já descreve sua função: Cascading Style Sheets. Ou seja, é uma linguagem usada para estilizar documentos web. Para isso, ele é referenciado no início do documento (head) e, através dos conceitos de classes e ID's ele determina estilos para os elementos HTML. Um ID pode ser utilizado apenas uma vez na página, mas a classe, por sua vez, não tem essa restrição. Desse modo, podemos definir regras para os elementos indicados pelas tags mostradas anteriormente, dispostas da seguinte maneira: seletores{declarações}. Sejam os elementos como seletores e as declarações idicando propriedade e valores para eles. Por exemplo:

  • Elemento em vermelho e maiúsculo

    • Aqui foram utilizados no elemento, as declarações:

    • text-transform: uppercase
    • color: red

  • Elemento sublinhado e grande

    • Nesse caso o elemento contém as regras de:

    • text-decoration: underline
    • font-size: 30px

Aqui temos poucos e básicos exemplos e uma breve introdução ao CSS. Através dessa liguagem é possível estruturar a página, determinar margens, alinhar o texto, definir bordas, cores de fundo, ...
